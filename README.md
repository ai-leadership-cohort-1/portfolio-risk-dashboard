# Small Business Loan Pre-Screening Prototype

An internal prototype for pre-screening small-business loan applications for
Australian retail banking. **Work in progress** — this README is a stub and
will be filled out (local run steps, Vercel deploy steps, and the exact file
to edit for scoring thresholds) as the build progresses.

> Prototype for internal pre-screening only. Not a credit decision. Full
> assessment required by the credit committee.
