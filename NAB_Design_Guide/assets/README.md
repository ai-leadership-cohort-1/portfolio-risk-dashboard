# NAB brand assets — 2024 "EVO" template

Brandmark extracted from `ppt/media/` of **NAB PPT Template_16x9_18Oct24**. Modern
NAB branding in this deck uses the **7‑point star alone** (no wordmark lockup).

| File | What it is | Use |
|------|------------|-----|
| `nab-star-red.png` | NAB star, red `#ED0000`, hi‑res (284×290) | Brandmark on white/light backgrounds |
| `nab-star-red-sm.png` | NAB star, red, small (170×173) | Cover top‑right corner mark |
| `nab-star-white.png` | NAB star, white (172×173) | Brandmark on RED / BLACK / image backgrounds |

Placement: top‑right corner of covers (~`8.94″, 0.29″` on the 10″ canvas, ~`0.77″`
square). Keep the star red on light backgrounds and white on dark/red backgrounds.

## Fonts (NOT bundled here)

Headings use **Epilogue Medium**, body uses **Source Sans Pro** (SemiBold for
emphasis). Both are free Google Fonts and were **not** embedded in the template.
Install them for PowerPoint, or load via Google Fonts / self‑hosted `@font-face`
for HTML:

- Epilogue — https://fonts.google.com/specimen/Epilogue
- Source Sans 3 (Source Sans Pro) — https://fonts.google.com/specimen/Source+Sans+3
